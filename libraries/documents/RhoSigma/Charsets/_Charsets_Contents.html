Codepage switching & UTF-8 generator

This library is for foreign QB64-PE users who need to use another codepage than the standard Pc437. While it's easy to switch the IDE codepage in the options, the made programs must always manually remap all 128 codepoints. This library provides a simple routine to perform the switch and do all required unicode remapping. The library also has various functions to convert any text or unicode sequences to UTF-8, ready to be used with QB64-PE's _UPRINT... functions.

Table of Contents

CODEPAGE SWITCHING

  • SetAnsiCodepage
    • set a new codepage and do required remapping
  • GetAnsiCodepage$
    • return the currently active codepage name

UNICODE CONVERSION

  • UnicodeToUtf8Char$
    • convert a single unicode into UTF-8 notation
  • Utf8CharLen
    • determine the byte length of a UTF-8 char
  • UnicodeStringToUtf8Text$
    • convert a string of unicodes into UTF-8 text
  • UnicodeArrayToUtf8Text$
    • convert an array of unicodes into UTF-8 text
  • UnicodeDataToUtf8Text$
    • convert hardcoded unicode DATAs into UTF-8 text
  • AnsiTextToUtf8Text$
    • convert any text into UTF-8 text using a certain codepage
