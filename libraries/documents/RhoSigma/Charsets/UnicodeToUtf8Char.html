UnicodeToUtf8Char$ (function)

Returns the UTF-8 char sequence for the given unicode codepoint. The result may be a one- or multiple byte sequence as required in UTF-8. The result is ready to be used with the various _UPRINT functions.

SYNTAX:

utf8char$ = UnicodeToUtf8Char$ (unicode&)

INPUTS:

unicode& (LONG)

  • The unicode codepoint to convert. See https://design215.com/toolbox/ascii-utf8.php for lookup

RESULT:

utf8char$ (STRING)

  • The UTF-8 notation for the given unicode codepoint.
